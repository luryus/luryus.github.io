Raspberry Pi Pico Macropad

I built a Raspberry Pi Pico based macropad for my own use. It has a completely custom firmware that handles the keyboard, rotary encoders, the user interface, and USB communication with the host. It works together with a Python-based server program that handles application-specific shortcuts and macros.
